# Surgical Video Segmentation
This code was tested on Ubuntu 22.04 with Python 3.10, Pytorch 2.1.0, and Cuda 11.8.0

## Setup Directory/Install Dependencies
```
bash setup.sh
```

## Running scipts
### Training 
```
bash train.sh
```

### Inference/Evaluation
```
bash inference.sh
bash evaluate.sh
```
